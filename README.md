*1.2 Objetivos Preliminares*  
O projeto visa capacitar pequenos empreendedores de empresas de pequeno porte que não possuem infraestrutura robusta de TI, focando nos seguintes pontos:  
- Identificação de Ameaças: Mapear os riscos digitais mais comuns e ataques cibernéticos voltados a pequenos comércios.  
- Prevenção de Baixo Custo: Propor soluções e comportamentos preventivos que não exijam altos investimentos financeiros.  
- Educação em LGPD: Traduzir os conceitos da Lei Geral de Proteção de Dados para uma linguagem acessível ao microempreendedor.  
- Material Didático: Produzir uma cartilha visual e prática para consulta rápida no dia a dia da empresa.

*2. DEFINIÇÃO DA PROPOSTA (ENTREGA 1)*  
*2.1 Problema a ser Resolvido*  
O Hotel Pousada Plaza Central SP armazena dados sensíveis de hóspedes (RG, CPF, dados bancários, endereço, e-mail) em sistemas e planilhas. A falta de políticas claras de segurança e conscientização dos funcionários expõe o negócio a riscos de vazamento, multas por descumprimento da LGPD e perda de confiança dos clientes.

*2.2 Objetivos*  
- Geral: Implementar práticas de segurança para proteger dados de hóspedes conforme LGPD.  
- Específicos:  
  - Identificar pontos críticos de risco (acesso a dados, backups).  
  - Criar código de conduta para funcionários sobre proteção de dados.  
  - Propor checklist de segurança para recepção e TI.  
  - Explicar LGPD de forma simples para funcionários.

Não inclui: sistemas complexos sem interface de entendimento fácil e gastos extras desnecessário

## 1. Sobre o Projeto
Cartilha de Segurança Digital focada em LGPD para funcionários da Pousada Plaza Central SP, um hotel de pequeno porte.

**Problema:** A pousada armazena dados sensíveis de hóspedes (RG, CPF, dados bancários, e-mail) em sistemas e planilhas. A falta de políticas claras e conscientização expõe o negócio a riscos de vazamento, multas da LGPD e perda de confiança dos clientes.

## 2. Objetivos
**Geral:** Implementar práticas de segurança para proteger dados de hóspedes conforme a LGPD.

**Específicos:**
- Identificar pontos críticos de risco: acesso a dados e backups
- Criar código de conduta para funcionários sobre proteção de dados
- Propor checklist de segurança para recepção e TI
- Traduzir conceitos da LGPD para linguagem acessível aos funcionários

## 3. Entregáveis
1. **Cartilha de Segurança Digital:** PDF com linguagem simples e visual
2. **Plano de Ação Simplificado:** Checklist prático para o dia a dia
3. **Código de Conduta:** Regras claras para funcionários

## 4. Escopo
**Inclui:** Material didático, checklist e código de conduta.
**Não inclui:** Implementação de sistemas complexos ou custos extras desnecessários.
